# 📱 Mobile Brand Detection

Detect and classify mobile phone brands using Deep Learning (EfficientNet).

## 🔥 Features

- Phone detection using OpenCV
- Brand classification (iPhone, Samsung, Vivo, Oppo)
- Confidence score display
- Clean UI bounding box output

## 🚀 Tech Stack

- Python
- PyTorch
- OpenCV
- PIL

## 📂 Structure

- uploads/ → input images
- results/ → output images

## ▶️ Run

```bash
python detect.py test.jpg
```
